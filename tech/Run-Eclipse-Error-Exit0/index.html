    • 解决eclipse运行时[exit code=1]错误

    Mon 16 October 2017

    问题

    Windows 10 64位操作系统，在运行Eclipse时出现如下信息。

    查了很多资料后，有人说是没有在eclipse的配置文件‘eclipse.ini’中指明JVM路径，需要加入：

    -vm
C:\Program Files (x86)\Java\jdk1.7.0_25\bin\javaw.exe

    但是实际情况是加入后仍然出现该问题，只是错误提示中的JVM路径改变为了设置路径。

    解决

    实际这个问题是因为eclipse是64位，而使用了32位的JVM。而实际在Oracle的默认jvm安装程序中，虽然计算机系统为64位，但是还是默认将虚拟机安装在了\Program Files (x86)\路径下。

    所以只需要在重新安装时，手动选择安装地址，将其安装在\Program Files\目录下即可。

    安装后可以在eclipse.ini中重新指定该路径，或者可以直接将这部分内容删除。

    现在eclipse应该可以正常运行。
